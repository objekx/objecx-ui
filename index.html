Welcome to Velin UI

A clean, lightweight, simple HTML-based design language for your website. No JavaScript framework needed.

What is Velin UI?

Velin UI is an HTML-first design system that gives you polished, consistent interface primitives without the baggage of JavaScript frameworks. Think Figma-level visual clarity with copy-paste code. Drop in a CSS file, use semantic classes, and get a coherent look across projects.
